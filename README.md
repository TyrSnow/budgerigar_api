# budgerigar_api
用于管理语言包的系统服务器
## 1. 名词解释
1. 项目 用于分隔的最小单位，一般情况下，我们认为前端、后台、app应该共享同一个项目
1. 语言包 用于输出的单位，一个项目可以拥有多个语言包，语言包默认是json格式，也可以使用自定义的模板来渲染键值对
1. 语句 将被翻译成多国语言的基本单元
1. 语句的相等 两条语句在去除前后的标点符号后一致
